# Synth
A basic synth (can plot wave).

## Keyboard commands
A -> DO

S -> RE

D -> MI

F -> FA

G -> SOL

H -> LA

J -> SI

K -> DO

L -> RE

## How to install

Open the terminal and navigate to the folder Synth, then type:

	$ make
	
## How to use

	$ su
	
	# ./synth

## Requirements

liballegro 5
	https://github.com/liballeg/allegro5
